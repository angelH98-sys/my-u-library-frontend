# My U Library - frontend app

Frontend app for My U Library, coded with ReactJs + TypeScript

## Run Locally

Clone the project

```bash
  git clone https://github.com/angelH98-sys/my-u-library-frontend.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
